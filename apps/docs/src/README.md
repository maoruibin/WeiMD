# WeiMD 文档

欢迎来到 WeiMD 文档中心。

WeiMD 是一款专为微信公众号设计的 Markdown 编辑器，支持一键复制到公众号、多种精美主题、深色模式预览等功能。 该项目 fork 自 [tenngoxars/WeMD: 更优雅的 Markdown 公众号编辑器](https://github.com/tenngoxars/WeMD)

## 快速开始

想立即体验？访问 [在线编辑器](https://weimd.gudong.site/)

### 新手用户

如果你是第一次使用 WeiMD：

* [用户快速上手](https://weimd.gudong.site/docs/quide/quick-start-user.html) — 5 分钟完成第一篇文章

### 开发者

如果你想参与开发或自定义：

* [开发者快速上手](https://weimd.gudong.site/docs/quide/developer-quickstart.html) — 本地环境搭建

## 核心功能

| 功能 | 说明 |
| :--- | :--- |
| **一键复制** | 完美兼容微信公众号 |
| **多主题** | 12+ 精美主题可选 |
| **深色模式预览** | 预览微信深色模式效果 |
| **本地优先** | 数据存储在本地，隐私安全 |

## 技术亮点

WeiMD 内置了 [微信深色模式预览算法](https://weimd.gudong.site/docs/reference/dark-mode-algorithm.html)，可在发布前预览文章在深色模式下的效果，还原度约 90%。
