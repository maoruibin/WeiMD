# 关于 WeMD

WeMD 是一个专注于微信公众号 Markdown 排版的工具。

## 核心理念

**"让写作回归纯粹，让排版更加优雅。"**

我们相信，优质的内容值得更好的呈现。WeMD 致力于为创作者提供最流畅的写作体验和最美观的排版效果。

## 主要特性

- **所见即所得**：实时预览微信排版效果，完美还原。
- **本地优先**：所有数据存储在本地，隐私安全无忧。
- **一键复制**：专为微信公众号优化，一键复制即可粘贴。
- **丰富主题**：内置多款精美主题，支持自定义 CSS。
- **多图床支持**：支持阿里云、腾讯云、七牛云等主流图床。
